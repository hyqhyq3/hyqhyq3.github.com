--- layout: post title: 在CentOS6 x86_64系统安装FMS4.5 --- 在CentOS6或者RedHat6上安装FMS4.5可能会遇到以下问题
  1. file: command not found
    2. yum install file
  2. 提示libcap未安装 vi installFMS，在这个文件里面找到/lib/libcap.so，将其改成/lib64/libcap.so
  3. 安装成功后，不能正常启动服务，使用netstat查看端口，发现没有监听1935 
    4. cd /lib64
ln -s libcap.so.1 libcap.so.2
